# tacotron2-zh

Experimental Chinese TTS based on Nvidia Tacotron2 for Pytorch

## Hardware Information

 - GPU : 3080, 10G
 - CPU : AMD Ryzen 7 5800X
 - Mem : 64G

## Environment

 - python : 3.9.12
 - pytorch : 1.12.1+cu116
 - GPU driver : 515.43.04
 - CUDA : 11.7
